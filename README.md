## Getting Started

### Prerequisites

Ensure you have installed:
- [Node.js (v18+)](https://nodejs.org/)
- npm or yarn
- BackendShrnk API running and accessible

### Installation

```bash
git clone https://github.com/Bombat18/FrontendShrink.git
cd FrontendShrink
npm install
```
cp .envEx .env
```
```
# Start the development server
npm run dev
```
